Clonar un Repositorio

Clonar un repositorio es el primer paso para trabajar en un proyecto en tu computadora local. Sigue estos pasos para clonar un repositorio desde GitHub: 

                
                    # Abre una ventana de línea de comandos o terminal
    
                    # Navega a la ubicación donde deseas clonar el repositorio
                    cd ruta/de/destino
    
                    # Ejecuta el comando git clone seguido de la URL del repositorio en GitHub
                    git clone https://github.com/tu-usuario/tu-repositorio.git
    
                    # Esto creará una copia local del repositorio en tu computadora
    
                    # Para acceder al directorio del repositorio clonado, puedes usar
                    cd tu-repositorio
    
                    # Ahora puedes trabajar en el proyecto y sincronizar tus cambios con GitHub

Creación de Pull Request desde Otra Rama

Una Pull Request (Solicitud de Extracción) te permite proponer cambios en un repositorio y solicitar que se fusionen en la rama principal. Sigue estos pasos para crear un Pull Request desde otra rama: 

                
                    # Clona el repositorio
                    git clone link_del_repositorio
    
                    # Ingresa a la carpeta clonada
                    cd nombre_de_la_carpeta_clonada
    
                    # Realiza cambios en tu rama
                    # (ejemplo: corrige un error y realiza un commit)
                    git commit -m "Corrección: Actualización de la rama"
    
                    # Envía los cambios desde tu rama a la rama principal del repositorio remoto
                    git push origin tu_rama:main -f
    
                    # En GitHub, crea un nuevo Pull Request
                    # y escribe "Close #NúmeroDeIssue" en el cuerpo del Pull Request
                    # para vincularlo automáticamente a una issue y cerrarla al fusionar

Eliminacion De Commits Desde Cierto Punto

Si necesitas eliminar commits desde un punto específico en la historia de tu repositorio, puedes seguir estos pasos en la línea de comandos. Asegúrate de tener acceso a Git y que estás en una ventana de línea de comandos con permisos de administrador (cmd admin): 

                
                    # Acceder a la carpeta del repositorio local
                    cd ruta/a/la/carpeta/de/tu/repositorio
    
                    # Clonar el último estado del repositorio desde GitHub
                    git clone UrlDeGitHub
    
                    # Entrar en la carpeta clonada
                    cd nombre-de-la-carpeta-clonada
    
                    # Ver el registro de commits y copiar el código del commit desde el que deseas resetear
                    git log
    
                    # Una vez seleccionado el código del commit, todos los commits encima de él se eliminarán
                    # Salir de la vista del registro de commits (presiona "Q" y luego Enter)
    
                    # Regresar a la ruta de la clonación
                    cd ruta/a/la/carpeta/de/tu/repositorio
    
                    # Realizar un reset duro (hard reset) al commit seleccionado
                    git reset --hard codigo_del_commit_seleccionado
    
                    # Confirmar el reset
                    git push -f origin main
